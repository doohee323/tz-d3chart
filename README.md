# tz-d3chart

multi-line and geographic D3 chart.

# Features
```
	1. multiple line
	2. brushing mini chart, integrate 3 charts (main / mini / map)
	3. clicking each legend's node, toggle the main chart's lines.
	4. left / right y axis
	5. show description for chart's line and map's circle 
	
```

# Install
```
	npm install
	bower install
	
```

# Run
```
	grunt serve
	
	http://localhost:8000/
	
```
